Create New Repository on Github

on project folder right click and select git bash here
echo "# repository name" >> README.md
git init git add README.md (optional)
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/iyusrustandi/repository name.git
git push -u origin main

on github select setting then pages
build and deployment source select "Deploy from a branch"
branch select main and save

if branch main not visible
on build and deployment source select github action
and back select deploy from a branch
if the are error on git bash "remote origin already exist"
type "git remote remove origin" ,
check with "git remote -v"
and start from git init again.

Clone and edit Repository with VS Code

- on github repository page click "Code"
- copy HTTPS url
- open terminal VS Code
- git config --global user.name "iyusrustandi"
- git config --global user.email "iyusrustandi2@gmail.com"
- git config --global --list

- on VS Code welcome page select "Clone Repository"
- paste link HTTPS url
- select folder to save repository clone
- click open
- to commit change select "Source Control" on sidebar image menu
- write message and click "Commit"
- click Sync Changes

Install tailwind on HTML

npm install -D tailwindcss npx tailwindcss init

tailwind.config.js
/** @type {import('tailwindcss').Config} */
module.exports = {
darkMode: 'class',
content: ['index.html'],
theme: {
extend: {},
},
plugins: [],
};

src/input.css
@tailwind base;
@tailwind components;
@tailwind utilities;

when build html
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch

HTML head
use tag link and href="src/output.css" rel="stylesheet"
